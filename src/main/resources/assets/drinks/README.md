Drink image assets
==================

Put optional drink PNG assets in this folder. POSView loads images by slugified menu name:

- Ca phe sua -> ca-phe-sua.png
- Bac xiu -> bac-xiu.png
- Matcha latte -> matcha-latte.png

If an image is missing, DrinkIconFactory falls back to a generated Swing icon, so the demo still runs.

Current demo images are copied from the Sothirich/CoffeeShopSystemManagement reference after the project owner confirmed they may be reused for this coursework demo.
